# 练习 05：购物车

## 目标

学习如何处理多个对象协作，以及如何在测试中处理外部依赖。

## 需求示例

- 可以加入商品
- 可以计算总价
- 可以应用优惠规则
- 可以输出订单摘要

## 建议

- 先把“计算总价”做成纯逻辑
- 再逐步加入优惠与外部依赖
- 优先测试行为，而不是内部调用次数
