# 练习 02：FizzBuzz

## 目标

练习如何把需求拆小，按 TDD 一步一步推进。

## 规则

- 输入 3 的倍数，返回 `"Fizz"`
- 输入 5 的倍数，返回 `"Buzz"`
- 输入 3 和 5 的公倍数，返回 `"FizzBuzz"`
- 其他数字返回数字本身

## 建议顺序

1. 先写 `1 -> 1`
2. 再写 `3 -> Fizz`
3. 再写 `5 -> Buzz`
4. 再写 `15 -> FizzBuzz`
5. 最后补其他数字
